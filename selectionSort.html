Selection Sort algorithm

The selection sort algorithm sorts an array by finding the smallest element from the unsorted portion of the array, and placing it at the end of the sorted section of the array. The algorithm uses two sub arrays within the given array. The worst case time complexity of selection sort is O(n^2), it is not effective for large data sets.

  • The sub array maintained at the start of the array is sorted from arr[0] to arr[i]
  • The other elements in the array are unsorted, elements arr[i +1] through arr[arr.length -1]

Each time the algorithm iterates through the unordered sub array, finding the smallest element and placing it at the end on the sorted sub array.

function selectionSort() {
for (let i = 0; i < arraySize; i++) {
let min = i;
for (let j = i + 1; j < arraySize; j++) {
if (divsHeight[j] < divsHeight[min]) {
min = j;
}
}
if (min !== i) {
let temp = divsHeight[i];
divsHeight[i] = divsHeight[min];
divsHeight[min] = temp;
}
} }
